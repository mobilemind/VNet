VNet Bookmark Setup

These four steps add a Pastelet bookmark to Mobile Safari that will paste in the User ID, check the “Agree to Terms” box and set input focus on the password field. If creating multiple pastelets, press reset in-between.

Step 1: Enter the text

Enter the User ID Email/Login to paste. Then tap Make Vnet Pastelet to reload this page for Step 2.

  



Pastelet as link: Pastelet

Step 2: Add a Bookmark

Tap the add button on the toolbar. Then select Add Bookmark

Take note of the name (or edit it), and then tap Save.

Step 3: Edit the Bookmark

Now edit the bookmark to turn it into a Pastelet. Tap the Bookmarks button in the toolbar.

Navigate to the bookmark you just saved. Do not select the bookmark.

Tap Edit to modify the list and then tap on the bookmark itself.

In the Edit Bookmark panel, tap in the field that begins with “http:”. You need to edit the start of the text, so tap and drag all the way to the left.

Place the edit point between the “?” and the “j”. Then tap delete until the edit point is at the start of the line. When done, the bookmark should look like:

Step 4: Use the Pastelet

When visiting a web page “paste” the Email/Login by selecting the Pastelet bookmark. Just tap the bookmarks icon.

Then tap on the Pastelet bookmark to “paste” its text into a form field.
