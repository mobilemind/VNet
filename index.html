VNet by mobilemind

bookmarklets to more quickly connect to the 'Net

The "VNet" bookmarklets make it easy to quickly sign-in and get access to the 'Net.

All of that is done without you doing copy-and-paste work or back-and-forth app switching.

Available versions include:

  • VNet - just user id and doesn't submit the 'Net form.
  • VNet+P - optimized to do everything
